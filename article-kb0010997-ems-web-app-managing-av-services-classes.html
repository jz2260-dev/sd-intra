EMS Web App: Managing AV Services for Classes

Class AV services, including Mediasite recording and microphones, are requested through the EMS Web App. Requests must be submitted at least 3 days before the class. Stern classes are recorded by default unless the instructor opts out.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the class reservation is approved in EMS.
  2. Confirm the request is at least 3 days before the class. For late requests, have the user email cetg@stern.nyu.edu with the reservation ID, course number, room number, date, time, and AV request details.
  3. If the request is for the same day as the class, have the user call the Stern IT Service Desk at 212-998-0180 to confirm whether it can be accommodated.
  4. Explain that Stern classes are recorded by default unless the instructor opts out.
  5. Have the user log in to the EMS Web App or select Reserve a Room from the Stern Life dashboard.
  6. Have the user select My Events from the left navigation.
  7. If the user is acting on behalf of someone else, have them select Act As from the menu under their name in the upper-right corner, then choose the appropriate user.
  8. If the target user does not appear, have the requester email helpdesk@stern.nyu.edu to request delegate access. Administrators may request access for faculty in their department. Teaching assistants should ask the instructor to send the request.
  9. On My Events, have the user select Add Services under Reservation Tasks, or Manage Services under the booking if a service has already been added.
  10. Have the user scroll to AV Services for Classes.
  11. To request Mediasite recording, have the user select Class Recording (Mediasite) under the recording section.
  12. Have the user select OK to save the change.
  13. To request a microphone, have the user return to AV Services for Classes and select the microphone link under Mics.
  14. Have the user review the default microphone quantity, which is set to 1.
  15. If another microphone is needed, have the user adjust the quantity. The maximum available for the classroom is shown as available inventory.
  16. Have the user select OK to save the change.
  17. After selecting all needed services, have the user select Next Step at the top of the page.
  18. To apply the same services to all class sessions, have the user select the checkbox in the heading on the next page.
  19. Have the user select Add Services in the upper-right corner to finalize the request.
  20. Explain that Mediasite presentation links are emailed to the instructor one hour before class and usually start working after the recording has been uploaded, processed, and converted into a presentation.
  21. If recordings are posted to Brightspace, explain that the user will receive a ServiceLink reference number and another email when the Brightspace site has been configured.
  22. To cancel services, have the user open My Events, select Cancel Services under Reservation Tasks, choose the service category, select the booking or bookings, and select Cancel Services.
