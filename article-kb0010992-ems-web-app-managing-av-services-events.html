EMS Web App: Managing AV Services for Events

Event AV services, including recording, microphones, projection, and technology setup assistance, are requested through the EMS Web App. Requests must be submitted at least 3 days before the event.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the event is approved in EMS. Services can only be added to approved events.
  2. Confirm the request is at least 3 days before the event. For late requests, have the user email ServiceDesk@stern.nyu.edu with the reservation ID, room number, date, time, and AV request details.
  3. Have the user log in to the EMS Web App or select Reserve a Room from the Stern Life dashboard.
  4. Have the user select My Events from the left navigation.
  5. If the user is acting on behalf of someone else, have them select Act As from the menu under their name in the upper-right corner, then choose the appropriate user.
  6. If the target user does not appear, have the requester email ServiceDesk@stern.nyu.edu to request delegate access. Administrators may request access for faculty in their department. Teaching assistants should ask the instructor to send the request.
  7. On My Events, have the user select Add Services under Reservation Tasks, or Manage Services under the booking if a service has already been added.
  8. Have the user scroll to Event Technology.
  9. Have the user complete required AV details for the booking.
  10. In Account Number, have the user enter the Chartfield number for the department managing the event.
  11. Have the user review Standard Technology and Wireless Mics for additional service options.
  12. For event technology requests, have the user select needed items and enter quantities for microphones and stands. For other items, a quantity of 1 is typically used.
  13. For Technology Setup Assistance, explain that someone from the team will check in before the event starts.
  14. For Mediasite Recording, have the user include start and stop times in the special instructions field. Recordings may be up to 4 hours. For longer events, request multiple recordings at least 5 minutes apart.
  15. Have the user complete the questionnaire at the end of the request form.
  16. Under Billing Information, have the user enter the Chartfield account number. Explain that this does not create immediate charges; the user will be contacted to approve any costs.
  17. For recurring events in the same space, have the user select the additional bookings and select SAVE.
  18. For recurring events in different rooms, advise the user to submit Event Technology requests individually to avoid errors.
  19. To cancel services, have the user open My Events, select Cancel Services under Reservation Tasks, choose the service category, select the booking or bookings, and select Cancel Services.
