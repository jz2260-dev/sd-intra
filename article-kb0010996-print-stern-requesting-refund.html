Print@Stern / PaperCut: Requesting a Refund

Users can request a refund for a print job through the Print@Stern web portal by opening Recent Print Jobs and selecting the refund request link for the affected job.

Support Workflow

Internal step path adapted from the source article.

  1. Have the user connect to the NYU wireless network.
  2. Have the user log in to PaperCut through a web browser using their NYU NetID and password.
  3. If the PaperCut Client is installed, the user can also select Details... from the small Print@Stern window that displays their balance.
  4. Have the user select Recent Print Jobs from the menu.
  5. Have the user find the job that needs a refund.
  6. Have the user select the request refund link on the right side of that job.
  7. Have the user enter a reason for the refund.
  8. Have the user select OK.
  9. Explain that a ticket will be created, the request will be reviewed, and a refund will be issued if approved.
