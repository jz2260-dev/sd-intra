Print@Stern / PaperCut: Printing an Attachment With Email to Print

Email to Print lets users print by sending supported file attachments from their Stern email account to a Print@Stern address. Color jobs go to color-print@stern.nyu.edu, and black-and-white jobs go to print@stern.nyu.edu.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the user is sending from their Stern email account. Messages must be sent from a Stern email address.
  2. For color printing, have the user send the message to color-print@stern.nyu.edu.
  3. For black-and-white printing, have the user send the message to print@stern.nyu.edu.
  4. Confirm the attachment type is supported. Supported files include Microsoft Excel, Microsoft Word, Microsoft PowerPoint, plain text, PDF, and common image formats such as JPEG and PNG.
  5. Have the user send the email with the file or files attached.
  6. Ask the user to wait for the response email confirming the job is being processed or explaining that it was rejected.
  7. If the job is processing, have the user check the job status in PaperCut User Client if needed.
  8. When the Print Job Notification appears in PaperCut User Client, have the user confirm the print job.
  9. Direct the user to a release station to finish printing.
