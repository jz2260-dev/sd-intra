Mobile Devices: Configure a Stern Account via Google Setup on iPhone, iPad, or iPod

Users can add their Stern account to iOS or iPadOS through Google account setup in Mail settings. They can choose which items to sync, such as mail, contacts, calendars, and notes.

Support Workflow

Internal step path adapted from the source article.

  1. Have the user open Settings from the device home screen.
  2. Have the user select Mail.
  3. Have the user select Mail Accounts.
  4. Have the user select Add Account. If other accounts are already configured, the user may need to scroll to find Add Account.
  5. Have the user enter their Stern email address in the format NetID@stern.nyu.edu.
  6. Have the user select Next.
  7. When prompted to choose a provider, have the user select Google.
  8. When prompted that Google requires authentication in a browser, have the user select Open Browser.
  9. On the NYU Single Sign On screen, have the user enter their username in the format NetID@nyu.edu.
  10. Have the user select Next.
  11. Have the user enter their NYU account password.
  12. Have the user select Sign In.
  13. If prompted for two-factor authentication, have the user choose and complete their preferred authentication method.
  14. If Google asks the user to choose an account to continue to iOS, have them select their Stern account.
  15. Have the user select Continue for the prompts that follow.
  16. Have the user choose sync options such as Mail, Contacts, Calendars, and Notes. Stern recommends syncing mail, calendar, and contacts.
  17. Have the user select Save.
