Mobile Devices: Google Account Setup on Android

Users can add their Stern account to Android through Google account setup. They can choose which account components to sync, such as email, calendar, and contacts.

Support Workflow

Internal step path adapted from the source article.

  1. Tell the user that menu labels may vary by Android device and version. These steps were documented using Android version 16.
  2. Have the user open Settings.
  3. Have the user select Accounts and Backup.
  4. Have the user select Accounts.
  5. Have the user select Add Accounts.
  6. Have the user select Google.
  7. On the Sign in screen, have the user enter their full Stern email address in the format Stern_ID@stern.nyu.edu.
  8. Have the user select Next.
  9. Have the user review Google's Privacy Policy and Terms of Service. If they agree, have them select Accept.
  10. On the NYU login page, have the user enter their NYU email address in the format NetID@nyu.edu.
  11. Have the user enter their NYU password.
  12. Have the user select Sign On.
  13. Have the user complete multi-factor authentication using their preferred method.
  14. After setup, have the user choose which account components to sync, such as email, calendar, and contacts.
