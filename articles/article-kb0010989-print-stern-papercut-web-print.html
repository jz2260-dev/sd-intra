Print@Stern / PaperCut: Printing From a Web Browser

PaperCut Web Print lets users upload documents through a web browser and send them to Print@Stern printers without installing print drivers. Users must be connected to the NYU network.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the user is connected to the NYU network.
  2. Have the user log in to the PaperCut User Portal with their Stern ID and Stern password.
  3. Have the user select Submit a Job >>.
  4. Have the user select the desired printer.
  5. Have the user select Print Options and Account Selection >>.
  6. Have the user choose print settings, such as double-sided or color printing.
  7. Have the user select Upload Documents >>.
  8. Have the user select Upload from computer.
  9. Have the user choose one or more files to upload.
  10. Have the user select Upload & Complete >>.
  11. Confirm the job status changes to Held in a queue.
  12. Direct the user to a release station or tap-to-release printer to print the job.
