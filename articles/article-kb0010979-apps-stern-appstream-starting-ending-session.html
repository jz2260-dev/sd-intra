Apps@Stern via AppStream: Starting and Ending a Streaming Session

Apps@Stern opens applications inside a secure browser-based Windows desktop called a streaming session. Users start a session from apps.stern.nyu.edu, launch an app or desktop view, and should fully end the session when finished.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the user is going to apps.stern.nyu.edu.
  2. Have the user select Go to Apps@Stern.
  3. Have the user sign in with their NYU email address in the format NetID@nyu.edu, then enter their NYU password.
  4. If the user is prompted for a role, have them select SSO-SternApps.
  5. On the catalog page, have the user select the application they need. If they need the full desktop view, have them select Desktop.
  6. Remind the user that the streaming session may take 1 to 2 minutes to start.
  7. To end the session, have the user hover over the user icon in the upper-right corner of the streaming session window.
  8. Have the user select End Session, then select End Session again to confirm.
  9. If the user immediately starts a new session after ending one, explain that AppStream may take 5 to 10 minutes to fully end the prior session.
