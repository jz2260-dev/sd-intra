Gmail@Stern: Requesting a Vanity Email Address

A Stern vanity email address is an alias for a Gmail@Stern account. It lets a user send and receive mail using an alternate Stern address, commonly in the format firstname.lastname@stern.nyu.edu. It does not replace the user's Stern ID or login username.

Support Workflow

Internal step path adapted from the source article.

  1. Explain that a vanity email address is an alias for Gmail@Stern and does not replace the user's Stern ID or Stern login username.
  2. Have the user log in to SIMON with their NYU email address.
  3. In SIMON, have the user open Account Info and select Vanity Email Address.
  4. Have the user review available vanity address options. If needed, have them select See More.
  5. Have the user select the address they want and select Submit.
  6. Tell the user that the address may take up to 24 hours to become active.
  7. If SIMON does not allow the user to request the desired address, have them email ServiceDesk@stern.nyu.edu.
  8. Ask the user to include their Stern ID, desired vanity email address, and a short reason for the request.
  9. If the user previously activated a vanity address, ask them to include this acknowledgement and replace the example address with their current vanity address: "I understand that by changing my vanity email address, I will no longer be able to use my current personal address, current.address@stern.nyu.edu, to send or receive email messages. Messages sent to this address will receive bouncebacks, or may be delivered to another Stern account if the email address is reassigned."
  10. If the user's desired vanity address depends on a name change, direct them to submit an official name change request with the NYU Registrar first.
  11. After activation, advise the user to test the vanity address from a non-Gmail@Stern account before sharing it.
  12. Once the vanity address is receiving mail, direct the user to set it up in Gmail using Send mail as if they want to send from that address.
