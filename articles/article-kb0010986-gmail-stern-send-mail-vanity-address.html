Gmail@Stern: Sending Mail From a Vanity Email Address in Gmail

After a Stern vanity email address is active and receiving messages, the user must add it to Gmail's Send mail as settings before they can send mail from that address.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the vanity address has already been activated.
  2. Confirm the vanity address can receive messages. If it was just activated, remind the user that activation can take up to 24 hours.
  3. Have the user sign in to Gmail@Stern using NYU credentials.
  4. Have the user select the gear icon near the upper-right corner.
  5. Have the user select See all settings.
  6. Have the user open the Accounts tab.
  7. In Send mail as, have the user select Add another email address you own.
  8. Have the user enter their display name in the Name field. Explain that this affects the name shown in outgoing messages, not the Stern directory.
  9. Have the user enter the vanity email address in the Email address field.
  10. Confirm Treat as an alias is selected.
  11. Have the user select Next Step.
  12. To send from the vanity address, have the user compose or reply to a message, select the address in the From field, and choose the vanity address.
  13. To make the vanity address the default, have the user return to Send mail as and select Make default for the vanity address.
