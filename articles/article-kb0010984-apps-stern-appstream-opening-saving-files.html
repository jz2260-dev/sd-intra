Apps@Stern via AppStream: Opening and Saving Files

Users can open and save files in Apps@Stern through temporary local session storage or by connecting Google Drive inside the AppStream desktop. Temporary Files is useful for moving files between the user's device and the session. Google Drive is better for files that need to remain available after the session ends.

Support Workflow

Internal step path adapted from the source article.

  1. Ask whether the user wants to use temporary local transfer or Google Drive.
  2. To upload from the user's local device, have the user select the My Files icon on the top AppStream toolbar.
  3. In My Files, have the user select Temporary Files as the destination folder.
  4. Have the user select Upload Files, choose the local file, and select Open.
  5. Have the user open Temporary Files from the AppStream desktop to confirm the upload completed.
  6. To open a file from Google Drive, have the user open the Google Drive app from the Apps@Stern catalog or desktop.
  7. Have the user select Get started, then Sign In.
  8. Have the user sign in with their NYU email address and verify the Google Drive app access request.
  9. On the NYU Single Sign On screen, have the user enter NetID@nyu.edu, enter their NYU password, and complete two-factor authentication if prompted.
  10. Have the user complete Google Drive sign-in and confirm that Google Drive is mapped to the virtual desktop.
  11. To work with a Google Drive file, have the user copy the file from Google Drive to the AppStream Documents folder, then open it with the needed application.
  12. To save a file to the user's local device, have the user select File > Save As in the Apps@Stern application and save to Temporary Files.
  13. Have the user open My Files, select Temporary Files, choose the saved file, select Actions, and select Download.
  14. To save to Google Drive, have the user select File > Save As, choose the mapped Google Drive location, and select Save.
  15. Have the user confirm the saved file from another device or browser after saving to Google Drive.
