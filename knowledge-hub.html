Apps@Stern via AppStream: Starting and Ending a Streaming SessionKB0010979 - Apps@Stern - Apps@Stern opens applications inside a secure browser-based Windows desktop called a streaming session. Users start a session from apps.stern.nyu.edu, launch an app or desktop view, and should fully end the session when finished. Gmail@Stern: Requesting a Vanity Email AddressKB0010981 - Email & Accounts - A Stern vanity email address is an alias for a Gmail@Stern account. It lets a user send and receive mail using an alternate Stern address, commonly in the format firstname.lastname@stern.nyu.edu. It does not replace the user's Stern ID or login username. Print@Stern / PaperCut: FAQKB0010983 - Printing - Print@Stern is a PaperCut-managed printing service for Stern students. Users send jobs to virtual printers, then release them at a Stern printer or release station. Credits are charged only when jobs are released. Apps@Stern via AppStream: Opening and Saving FilesKB0010984 - Apps@Stern - Users can open and save files in Apps@Stern through temporary local session storage or by connecting Google Drive inside the AppStream desktop. Temporary Files is useful for moving files between the user's device and the session. Google Drive is better for files that need to remain available after the session ends. Gmail@Stern: Sending Mail From a Vanity Email Address in GmailKB0010986 - Email & Accounts - After a Stern vanity email address is active and receiving messages, the user must add it to Gmail's Send mail as settings before they can send mail from that address. Print@Stern / PaperCut: Printing From a Web BrowserKB0010989 - Printing - PaperCut Web Print lets users upload documents through a web browser and send them to Print@Stern printers without installing print drivers. Users must be connected to the NYU network. Print@Stern / PaperCut: Locating a Printer and Release StationKB0010990 - Printing - Print@Stern printers are located throughout NYU Stern buildings. Undergraduate printers are primarily in Tisch Hall. Graduate printers are in Tisch Hall, Kaufman Management Center, and Shimkin Hall. Print@Stern / PaperCut: Printing an Attachment With Email to PrintKB0010991 - Printing - Email to Print lets users print by sending supported file attachments from their Stern email account to a Print@Stern address. Color jobs go to color-print@stern.nyu.edu, and black-and-white jobs go to print@stern.nyu.edu. EMS Web App: Managing AV Services for EventsKB0010992 - Classroom & Event Technology - Event AV services, including recording, microphones, projection, and technology setup assistance, are requested through the EMS Web App. Requests must be submitted at least 3 days before the event. EMS Web App: Automatic Posting of Mediasite Content to BrightspaceKB0010993 - Classroom & Event Technology - Automatic posting of Mediasite recordings to Brightspace is requested in EMS by editing the class reservation details and setting the Brightspace posting option to Yes. An active recording request is required. Mobile Devices: Configure a Stern Account via Google Setup on iPhone, iPad, or iPodKB0010994 - Mobile Devices - Users can add their Stern account to iOS or iPadOS through Google account setup in Mail settings. They can choose which items to sync, such as mail, contacts, calendars, and notes. EMS Web App: Adding Services to an Approved BookingKB0010995 - Classroom & Event Technology - After a booking is approved in EMS, users may be able to add catering and AV services. Print@Stern / PaperCut: Requesting a RefundKB0010996 - Printing - Users can request a refund for a print job through the Print@Stern web portal by opening Recent Print Jobs and selecting the refund request link for the affected job. EMS Web App: Managing AV Services for ClassesKB0010997 - Classroom & Event Technology - Class AV services, including Mediasite recording and microphones, are requested through the EMS Web App. Requests must be submitted at least 3 days before the class.
