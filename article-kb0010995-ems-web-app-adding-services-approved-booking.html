EMS Web App: Adding Services to an Approved Booking

After a booking is approved in EMS, users may be able to add catering and AV services. EMS may require one initial service to be added before additional services such as catering or AV can be managed.

Support Workflow

Internal step path adapted from the source article.

  1. Confirm the booking is approved.
  2. Have the user open the EMS Web App.
  3. Have the user select My Events from the left navigation.
  4. Have the user locate the approved reservation or booking.
  5. Have the user select Add Services under Reservation Tasks on the right side of the browser window.
  6. On Select Services, have the user select one service to add to the booking. Explain that this lets EMS mark the booking as having at least one service, and additional services can be added later.
  7. Have the user check I have read and agree to the terms and conditions.
  8. Have the user select Next Step.
  9. Have the user select the booking where the service should be added.
  10. Have the user select Add Services.
  11. Confirm that Manage Services now appears for the booking on the My Events page.
  12. To add catering, have the user select Add Services or Manage Services, scroll to Catering Menu, add the desired catering items, select Next Step, select the bookings, and select Add Services.
  13. Remind the user that catering requests must be submitted at least 7 days before the event.
  14. To add AV services, have the user select Add Services or Manage Services, scroll to Event Technology, complete the required AV information, enter the Chartfield account number, and select additional services under Standard Technology or Wireless Mics.
  15. Remind the user that AV service requests must be submitted no later than 3 days before the event.
  16. To cancel services, have the user open My Events, select Cancel Services under Reservation Tasks, choose the service category, select the booking or bookings, and select Cancel Services.
